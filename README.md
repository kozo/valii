## Setup

```
composer require fusic/valii
```

## validation

- Tel
  - 電話番号フォーマットチェック
- Hiragana
  - ひらがなのみチェック
- Katakana
  - カタカナのみチェック
- HankakuKatakana
  - 半角ｶﾀｶﾅチェック
- Zenkaku
  - 全角のみチェック
- ZipCode
  - 郵便番号チェック
- test

## Usage

```
public function rules()
    {
        return [
            "name" => 'required|hiragana',
            "tel" =>  'tel',
            "zip_code"  => [
                'zip_code'
            ],
            "strict_zip_code"  =>  [
                'required',
                'zip_code:strict'
            ]
        ];
    }
```
